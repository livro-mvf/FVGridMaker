SOLID e C++ moderno
===================

Arquivo criado para registrar como a FVGridMaker usará o espírito do SOLID sem
adotar herança ou classes virtuais como mecanismo principal.

Função esperada: documentar responsabilidade única, baixo acoplamento, interfaces
pequenas, composição, concepts, traits e policies como alternativas modernas para
manter o código simples, testável e eficiente.
