FVGridMaker
===========

FVGridMaker é uma biblioteca C++20 para construir malhas estruturadas de
volumes finitos. A biblioteca fornece eixos 1D, composição tensorial 2D,
medidas geométricas dependentes do sistema de coordenadas e exportação CSV/VTK.

O projeto privilegia valores sempre válidos, composição, ``concepts`` e
armazenamento contíguo. A biblioteca de produção depende apenas da biblioteca
padrão do C++.
